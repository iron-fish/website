## Check your balance

You can check your balance at any time by running the following command:

```sh
ironfish wallet:balance
```

Your wallet must be synced with the blockchain or it will be wrong. You can ask for a balance explanation by running

```sh
ironfish wallet:balance --explain
```

```sh
ironfish wallet:send
```

## Receive a transaction
To receive a transaction, give the sender your public key by running the following command:

```sh
ironfish wallet:address
```

## Track a transaction

Once a transaction is sent, the sender should see a transaction hash displayed.

```sh
Transaction Hash: 4f484a5355a65be2179fc66d69e149788e78aec35c6db230b182264e3982bc15
```

You can use the [Iron Fish Block Explorer](https://explorer.ironfish.network/) to display information about this transaction. Of course, because Iron Fish is a fully private cryptocurrency, a limited amount of information is available to the public.

For example, see the transaction above you can go here [https://explorer.ironfish.network/transaction/4f484a5355a65be2179fc66d69e149788e78aec35c6db230b182264e3982bc15](https://explorer.ironfish.network/transaction/4f484a5355a65be2179fc66d69e149788e78aec35c6db230b182264e3982bc15).
