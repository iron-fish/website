Bootstrap nodes are special nodes that you connect to to get information about new peers. By default there is an Iron Fish foundation provided bootstrap node for each network.

#### Using a custom bootstrap node

To add a new bootstrap node you need to use `ironfish config:set <boostrap1>,<bootstrap2>` command. See the example below.

```sh
$ ironfish config:set bootstrapNodes 1.foo.com,2.foo.com

$ ironfish config:get bootstrapNodes
1.foo.com,2.foo.com
```

> **Note:** `ironfish config:set` will replace any existing boostrap nodes rather than append new ones. You can also use `ironfish config:edit` to edit your config manually.
