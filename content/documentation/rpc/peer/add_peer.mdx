Attempts to connect to a peer through WebSockets using the given host and port. If whitelist is true (by default) the node will continue to attempt connections even if the initial connection fails

#### Request

```js
{
  host: string
  port?: number
  whitelist?: boolean
}
```

#### Response

```js
{
  added: boolean
}
```
