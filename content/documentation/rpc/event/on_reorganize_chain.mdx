---
title: event/onReorganizeChain
description: RPC Event | Iron Fish Documentation
---

Streams block headers when the chain reorganizes from a fork. `oldHead` is the
head of the chain before the reorg, `newHead` is the head of the chain after the
reorg, and `fork` is the header for the block at which the two forks diverged.

#### Request Body

```js
undefined
```

#### Response

```js
{
  oldHead: {
    hash: string
    sequence: number
    previousBlockHash: string
    timestamp: number
    difficulty: string
    graffiti: string
  }
  newHead: {
    hash: string
    sequence: number
    previousBlockHash: string
    timestamp: number
    difficulty: string
    graffiti: string
  }
  fork: {
    hash: string
    sequence: number
    previousBlockHash: string
    timestamp: number
    difficulty: string
    graffiti: string
  }
}
```
