Streams serialized transactions on gossip events

#### Request Body

```js
undefined
```

#### Response

```js
{
  serializedTransaction: string
}
```

### Example
```bash
# Request
curl -X POST -N http://localhost:8021/event/onTransactionGossip

# Response
{
  "data": {
    "serializedTransaction": "01010000...e931fd04"
  }
}
```
