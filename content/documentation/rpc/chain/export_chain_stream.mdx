Exports the chain as a stream of blocks.

If the start and stop sequences are provided, only the blocks between the start and
stop will be returned.

#### Request

```ts
{
  start?: number | null
  stop?: number | null
} | undefined
```

#### Response

```ts
{
  start: number
  stop: number
  block?: RpcBlockHeader & {
    main: boolean
    head: boolean
    latest: boolean
    /**
     * @deprecated Please use sequence instead
     */
    seq: number
    /**
     * @deprecated Please use previousBlockHash instead
     */
    prev: string
  }
}
```
