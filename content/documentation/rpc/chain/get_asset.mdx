Gets an asset from the blockchain from a given identifier

#### Request

```js
{
  id: string;
}
```

#### Response

```js
{
  createdTransactionHash: string;
  id: string;
  metadata: string;
  name: string;
  owner: string;
  supply: string;
}
```
