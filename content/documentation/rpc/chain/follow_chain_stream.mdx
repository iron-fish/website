Follows the chain from a given sequence and streams blocks from chain connects
and disconnects. If `wait` is set to false the stream will end when it reaches
the head of the chain (true by default).

#### Request

```ts
{
  head?: string | null
  serialized?: boolean
  wait?: boolean
} | undefined
```

#### Response

```ts
{
  type: 'connected' | 'disconnected' | 'fork'
  head: {
    sequence: number
  }
  block: RpcBlock
}
```

RPC Objects: 
- [RpcBlock](/developers/documentation/rpc/objects/rpcBlock)
