Unlocks the accounts in the wallet with a given passphrase.

Throws an error if the passphrase is invalid. If the wallet is already unlocked, it renews the timer.

The default timeout is 24 hours.

#### Request

```ts
{
  passphrase: string
  timeout?: number
}
```

#### Response

```ts
undefined
```
