title: wallet/getBalances
description: RPC Wallet | Iron Fish Documentation
---

Gets the wallet's `$IRON` balance, as well as balances of custom assets of the given account. If the account is not specified, the default account will be used.

#### Request

```js
{
  account?: string
  confirmations?: number
}
```

#### Response

```js
{
  account: string
  balances: Array<{
    assetId: string
    assetName: string
    assetOwner: string
    assetVerification: {
      status: 'verified' | 'unverified' | 'unknown'
    }
    confirmed: string
    unconfirmed: string
    unconfirmedCount: number
    pending: string
    pendingCount: number
    available: string
    blockHash: string | null
    sequence: number | null
  }>
}
```
