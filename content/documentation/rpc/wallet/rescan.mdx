Resets all accounts and rescans them for transsactions

#### Request

```ts
{
  /**
   * If true, the RPC will stream back the progress of the rescan
   */
  follow?: boolean
}
```

#### Response

```ts
{
  sequence: number;
  startedAt: number;
  endSequence: number;
}
```
