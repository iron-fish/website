Creates a transaction, posts the transaction, and submits it to the wallet, mempool, and network.

#### Request

```js
{
  account?: string
  outputs: {
    publicAddress: string
    amount: string
    memo: string
    assetId?: string
  }[]
  fee?: string
  feeRate?: string
  expiration?: number | null
  expirationDelta?: number | null
  confirmations?: number | null
}
```

#### Response

```js
{
  account: string;
  hash: string;
  transaction: string;
}
```
