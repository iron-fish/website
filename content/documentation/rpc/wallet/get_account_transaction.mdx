Gets a transaction for an account. If the account is not specified, the default account is used.

#### Request

```ts
{
  hash: string
  account?: string
  serialized?: boolean
  confirmations?: number
  notes?: boolean
  spends?: boolean
}
```

#### Response

```ts
{
  account: string
  transaction: RpcWalletTransaction | null
}
```

RPC Objects:
- [RpcWalletTransaction](/developers/documentation/rpc/objects/rpcWalletTransaction)
