Aggregate signature shares from participants to sign a transaction. If the broadcast flag is passed, the transaction will be broadcast to the network.

#### Request

```ts
{
  account?: string
  signingPackage: string
  signatureShares: Array<string>
  broadcast?: boolean
}
```

#### Response

```ts
{
  accepted: boolean
  broadcasted: boolean
  transaction: string
}
```
