title: wallet/multisig/createSignatureShare
description: RPC Multisig | Iron Fish Documentation
---

Creates a signature share for a participant for a given signing package. These signatures are then collected and used by the aggregateSignatureShares RPC to create the final signed transaction.

#### Request

```ts
{
  account?: string
  signingPackage: string
}
```

#### Response

```ts
{
  signatureShare: string
}
```
