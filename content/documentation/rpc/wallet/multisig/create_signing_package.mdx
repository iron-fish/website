Creates a signing package for a given transaction for a multisig account. After this step, the created signing package will be used by the participants to create a signature share using the RPC [wallet/multisig/createSignatureShares](/developers/documentation/rpc/wallet/multisig/create_signature_shares).

#### Request

```ts
{
  unsignedTransaction: string
  commitments: Array<string>
  account?: string
}
```

#### Response

```ts
{
  signingPackage: string
}
```
