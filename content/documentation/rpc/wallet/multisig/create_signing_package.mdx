Creates a signing package for a given transaction for a multisig account. The signing package will be used by the participants to create a signature share using the RPC createSignatureShare.

#### Request

```ts
{
  unsignedTransaction: string
  commitments: Array<string>
  account?: string
}
```

#### Response

```ts
{
  signingPackage: string
}
```
