Creates the siging commitment for a participant of a multisig wallet. This means that the participant will participate in the signing process. These commitments will be aggregated from all participants and used in the [wallet/multisig/createSigningPackage](developers/documentation/rpc/wallet/multisig/create_signing_package) to create the signing package. 

#### Request

```ts
{
  account?: string
  unsignedTransaction: string
  signers: Array<{ identity: string }>
}
```

#### Response

```ts
{
  commitment: string
}
```
