Creates a transaction burning a custom asset from a given account, posts the transaction, and submits it to the wallet, mempool, and network.

#### Request

```ts
{
  account?: string
  assetId: string
  fee?: string
  feeRate?: string
  value: string
  expiration?: number
  expirationDelta?: number
  confirmations?: number
}
```

#### Response

```ts
RpcBurn & {
  asset: RpcAsset
  transaction: RpcWalletTransaction
}
```

RPC Objects: 
- [RpcBurn](/developers/documentation/rpc/objects/rpcBurn)
- [RpcAsset](/developers/documentation/rpc/objects/rpcAsset)
- [RpcWalletTransaction](/developers/documentation/rpc/objects/rpcWalletTransaction)
