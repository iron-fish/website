> **Note:** This endpoint only works when using Testnet

Submits a request to get funds from the faucet for an account.

If an account is not provided, the default account on the node will be used.

#### Request

```ts
{
  account?: string
  email?: string
}
```

#### Response

```ts
{
  id: string;
}
```
