Use this command to sign an unsigned transaction. You do not need to specify the account, the RPC will iterate through all accounts to find the one that can sign the transaction. You can produce an unsigned transaction by running [`ironfish wallet:send`](/use/get-started/cli-cmd-wallet-send) / [`ironfish wallet:mint`](/use/get-started/cli-cmd-wallet-mint) / [`ironfish wallet:burn`](/use/get-started/cli-cmd-wallet-burn) with the `--unsignedTransaction` flag.

<Terminal command="ironfish wallet:sign"
  output={`

You can post the transaction to the Ironfish network either using the `--broadcast` flag or you can use the [`ironfish wallet:transaction:add`](/use/get-started/cli-cmd-wallet-transaction-add) command to add the signed transaction to your wallet.




# Flags

| Flag | Description | Default |
| ---- | ----------- | ------- |
| `-u, --unsignedTransaction` | Unsigned transaction to sign. ||
| `--ledger` | Sign with a ledger device | false |
| `--broadcast` | Broadcast the transaction to the network after signing | false |
| `--watch` | Wait for the transaction to be confirmed. If you use `--watch` you must also use `--broadcast` | false |
