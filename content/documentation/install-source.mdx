## Install

First, head over to GitHub to clone our [official repository](https://github.com/iron-fish/ironfish). Once that's done, follow the initial setup steps in the [README](https://github.com/iron-fish/ironfish#initial-setup) to install from source.

Afterwards, head to the [CLI README](https://github.com/iron-fish/ironfish/tree/master/ironfish-cli#use-scenarios) to run the CLI.
